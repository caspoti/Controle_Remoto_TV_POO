## Projeto de simulação de controle remoto de TV





### Tecnologias usadas:

* ##### Python
* #### Rich\_ (interface terminal)
* ##### Programação orientada a objetos (POO)





### Comandos aceitos:



* #### Ligar/Desligar TV: '@'
* #### Mudar de canal: '<'  '>'
* #### Mudar volume: '-' '+'
* #### Encerrar programa: '0'





Observação: Em alguns sistemas as cores do biblioteca Rich\_ podem não aparecer, ative a opção 'Emulate terminal in output console'.
